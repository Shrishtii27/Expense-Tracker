## 📖 Overview

**Expense Tracker** is a full-stack financial management application that empowers users to record, organize, and analyze their daily transactions. With real-time tracking, categorized expenses, and insightful visual analytics, users can make smarter financial decisions and maintain better control over their budget.

---

## ✨ Key Features

### 📊 Real-Time Expense Tracking
- Record income and expenses instantly.
- View updated balances in real time.
- Maintain an accurate financial overview.

### 🏷️ Smart Categorization
- Organize transactions into custom categories.
- Separate income and expenses efficiently.
- Improve budgeting and spending analysis.

### 📈 Interactive Analytics
- Visualize spending patterns using dynamic charts.
- Monitor financial trends over time.
- Gain actionable insights from transaction data.

### 🎨 User-Friendly Interface
- Clean and responsive design.
- Seamless navigation across devices.
- Optimized user experience for efficient management.

### 📄 Export Transaction History
- Download transaction records for reporting and bookkeeping.
- Maintain personal financial archives with ease.

## 🛠️ Tech Stack

### Frontend
| Technology | Purpose |
|------------|----------|
| React.js | User Interface Development |
| HTML5 | Application Structure |
| CSS3 | Styling & Responsive Design |
| JavaScript (ES6+) | Client-Side Functionality |

### Backend
| Technology | Purpose |
|------------|----------|
| Node.js | Server Runtime |
| Express.js | REST API Development |

### Database
| Technology | Purpose |
|------------|----------|
| PostgreSQL | Relational Database Management |

### Libraries & Tools
| Tool | Usage |
|-------|--------|
| Chart.js | Data Visualization |
| Mongoose* | Database Modeling |
| Vercel | Frontend Deployment |
| Supabase | Database & Backend Services |

> **Note:** Mongoose is typically used with MongoDB. If PostgreSQL is the primary database, consider using Prisma, Sequelize, or TypeORM instead.
